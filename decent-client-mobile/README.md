# decent-client-mobile

Mobile DecentChat client scaffold for iOS, implemented with Svelte 5 and Capacitor.

## Purpose

`decent-client-mobile` provides the mobile UI shell and native iOS packaging layer for DEP-016.

## Tech stack

- Svelte 5 (mobile UI)
- Vite (frontend tooling)
- Capacitor (native app shell)

## Planned capabilities

- Bottom-tab navigation: Calls, Workspaces, Chats, You
- Integration with `decent-client-core` for shared cross-platform logic
- iOS lifecycle and plugin integration via Capacitor

## Status

Initial scaffold only. Functional messaging/calls flows will be added in subsequent phases.
