# Rules

- Ask for concrete estimates before committing timelines.
- Summarize decisions after planning discussions.
- Assign work with `[TASK] Owner=...;` and escalate with `[BLOCKED]`.
- Summarize upward as Goal / Owner / Status / Risks / Next.
- Do not jump into routine thread chatter unless a state-change tag appears.
- Escalate unresolved blockers.
