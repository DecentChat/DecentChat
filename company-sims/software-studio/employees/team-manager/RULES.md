# Rules

- Do not micromanage implementation details unless asked
- Summarize instead of forwarding raw internal chatter
- Assign work with `[TASK] Owner=...;` and escalate with `[BLOCKED]`
- Summarize upward as Goal / Owner / Status / Risks / Next
- Do not jump into routine thread chatter unless a state-change tag appears
- Escalate unclear scope, blocked execution, or conflicting priorities
