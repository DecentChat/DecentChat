Under the hood

DecentChat is built on real cryptographic primitives — the same foundations as Signal. No security theater.

  • X25519 Diffie-Hellman key exchange
  • Double Ratchet with AES-256-GCM
  • BIP39 mnemonic for deterministic key generation
  • WebRTC data channels for P2P transport
  • Forward secrecy — compromised keys can't decrypt past messages
protocol.ts 
// Generate your identity from a seed phrase
const seed = generateMnemonic();
// "ocean drum capable antenna rally..."

// Derive crypto keys deterministically
const identity = await deriveKeys(seed);

// Create an invite for a friend
const invite = createInvite({
  publicKey: identity.publicKey,
  signal: await createOffer()
});

// Connect P2P — messages are encrypted
// with the Double Ratchet protocol
const channel = await connect(invite);
channel.send("hey, no servers here 🐙");
